# 🌧️ Weather Trend And Climate Anomaly Analysis Using ML

## 📌 Project Overview

This project focuses on analyzing historical rainfall data from 2000 to 2025 using Data Analytics and Machine Learning techniques. The objective is to identify rainfall patterns, seasonal variations, yearly trends, and rainfall variability while also developing a simple prediction model for future rainfall forecasting.

The project applies data preprocessing, exploratory data analysis (EDA), visualization, and Linear Regression to transform raw rainfall data into meaningful insights.

---

## 🎯 Objectives

* Analyze historical rainfall data.
* Study monthly, seasonal, and yearly rainfall patterns.
* Identify rainfall trends and variability.
* Determine wettest and driest years.
* Visualize rainfall distribution using graphs.
* Develop a rainfall prediction model using Linear Regression.
* Evaluate model performance using regression metrics.

## 📊 Key Findings

* August is the rainiest month.
* December is the driest month.
* 2010 was identified as the wettest year.
* 2018 was identified as the driest year.
* Monsoon season contributes the majority of annual rainfall.
* Rainfall exhibits significant year-to-year variability.
* Linear Regression captures the general rainfall trend but struggles with extreme fluctuations.
